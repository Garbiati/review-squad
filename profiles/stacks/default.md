# Profile: Default (Generic)

## When to Use
Used when the repository has no specific profile mapped and auto-detection could not determine one.

## Generic Review Checklist

### Code
- [ ] Clean and readable code
- [ ] Descriptive names for variables and functions
- [ ] No duplicated code (DRY)
- [ ] No dead or commented-out code
- [ ] Functions with single responsibility
- [ ] Acceptable cyclomatic complexity

### Security
- [ ] No hardcoded secrets
- [ ] Input validation
- [ ] Parameterized queries
- [ ] No obvious OWASP Top 10 vulnerabilities

### Quality
- [ ] Tests for new code
- [ ] Edge cases considered
- [ ] Proper error handling
- [ ] Adequate logging
