# Profile: Vue.js (Frontend Web)

## Tech Stack
- **Vue 2/3** (check version)
- **Vuex** / **Pinia** for state management
- **Vue Router**
- **TypeScript** or **JavaScript**

## Review Checklist

### Vue
- [ ] Well-structured Single File Components
- [ ] Composition API (Vue 3) vs Options API (Vue 2)
- [ ] Props validated with types
- [ ] Emitted events documented
- [ ] Computed properties vs methods vs watchers (proper usage)

### Performance
- [ ] Lazy loading of routes
- [ ] v-show vs v-if (proper usage)
- [ ] Keys in v-for
- [ ] Avoid unnecessary deep watchers
